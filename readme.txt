== Description ==

Display lists of your posts on any page or post using a simple shortcode

[wt4_recent_posts]

The default number of posts is set in settings > reading > Blog pages show at most

To change the number of posts for a single list use the following:

[wt4_recent_posts posts="4"]

To Display posts from a category use:

[wt4_recent_posts cat="test"]

To display posts per tag use:

[wt4_recent_posts tag="test"]

The shortcodes can be merged, to display 5 posts from test category use:

[wt4_recent_posts cat="test" posts="5" ]

Demo http://wpthemes4u.co.uk/plugins/easy-recent-posts/

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
